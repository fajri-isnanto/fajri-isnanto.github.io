Longhorn adalah distributed block storage system yang ringan dan mudah dioperasikan untuk Kubernetes. Mendukung snapshot, backup, disaster recovery, dan CSI-compliant.

## Requirement
- Kubernetes cluster
- Node dengan disk tersedia
- open-iscsi terinstall di semua node

## Ringkasan
Longhorn berhasil dideploy dengan beberapa StorageClass untuk tiering storage, siap melayani PVC workload stateful.
