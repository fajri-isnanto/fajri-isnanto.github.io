Velero adalah tool open-source untuk backup, restore, dan migrasi resource Kubernetes serta persistent volume. Integrasi dengan S3-compatible storage untuk off-cluster backup.

## Requirement
- Kubernetes cluster
- S3-compatible storage (MinIO)
- Velero CLI

## Ringkasan
Velero berhasil dikonfigurasi dengan scheduled backup harian, siap untuk disaster recovery scenario dengan RTO yang terukur.
