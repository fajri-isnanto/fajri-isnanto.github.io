kube-prometheus-stack menyediakan monitoring end-to-end: Prometheus untuk metrics collection, Alertmanager untuk alert routing, dan Grafana dengan pre-built dashboards untuk visibility cluster.

## Requirement
- Kubernetes cluster
- Helm v3
- StorageClass untuk persistent metrics

## Ringkasan
Monitoring stack lengkap berjalan dengan alert rules untuk node, pod, dan service. Dashboard Grafana siap untuk operational visibility.
