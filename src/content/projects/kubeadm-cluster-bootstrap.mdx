Proses bootstrap cluster Kubernetes menggunakan kubeadm meliputi instalasi containerd, kubeadm, kubelet, dan kubectl di semua node. Dilanjutkan dengan inisialisasi control-plane menggunakan `kubeadm init` dan join worker node dengan token yang dihasilkan.

## Requirement
- OS Ubuntu Server 24.04 LTS
- Minimal 2 GB RAM, 2 CPU
- Koneksi internet
- Container runtime (containerd)

## Ringkasan
Cluster Kubernetes vanilla berhasil di-bootstrap dengan kubeadm, siap untuk deployment workload dan integrasi dengan CNI, CSI, dan komponen infrastruktur lainnya.
