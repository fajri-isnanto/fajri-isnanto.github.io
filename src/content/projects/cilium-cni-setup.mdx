Cilium menyediakan networking, observability, dan security berbasis eBPF tanpa overhead iptables. Deploy menggunakan Helm dengan konfigurasi kube-proxy replacement dan Hubble UI untuk visibility traffic.

## Requirement
- Kubernetes cluster sudah berjalan
- Helm v3 terinstall
- Kernel 5.10+

## Ringkasan
Cilium berhasil dideploy sebagai primary CNI dengan Hubble observability, siap untuk implementasi NetworkPolicy L7 dan service mesh.
