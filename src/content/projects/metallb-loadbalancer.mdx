MetalLB adalah implementasi load balancer untuk cluster Kubernetes yang berjalan di bare-metal tanpa cloud provider. Mode L2 menggunakan ARP/NDP untuk mengumumkan IP service ke jaringan lokal.

## Requirement
- Kubernetes cluster tanpa cloud provider
- Subnet IP yang tersedia untuk pool load balancer

## Ringkasan
MetalLB berhasil dikonfigurasi, service type LoadBalancer kini mendapatkan external IP dari pool yang ditentukan.
